## Quick start

Backend

1. cd backend
2. npm install
3. cp .env.example .env # fill in MONGO_URI and JWT_SECRET
4. npm run dev # http://localhost:5000

Frontend

1. cd frontend
2. npm install
3. npm run dev # http://localhost:5173

Tip: If your API isn‚Äôt on localhost:5000, set VITE_API_BASE_URL in frontend `.env`.
